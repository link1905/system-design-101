In the final topic,
we'll discuss about common patterns in different facets to develop a system:

**[1. Event-Driven Architecture]({{< ref "event-driven-architecture" >}})**:

- Learn basically about the **Event-Driven Architecture**.
- Learn **Event Sourcing** and **Command-Query Segregation** patterns to effectively build an event-driven system.

**[2. Distributed Transaction]({{< ref "distributed-transaction" >}})**:

- Learn about low-level **blocking protocols** to execute distributed transactions.
- Learn about **compensation protocols** to execute distributed transactions asynchronously.

**[3. Caching Patterns]({{< ref "caching-patterns" >}})**:

- Learn caching patterns and techniques to implement a high-performant system.

**[4. Distributed Job Patterns]({{< ref "distributed-job-patterns" >}})**:

- Discuss basic models (**Streaming**, **Batching**, **Kappa**) to implement a cost-effective job system.
- Learn to model an asynchronous job system with **Directed Acyclic Graph (DAG)**.
